# @devlaunchkit/web

The Next.js 15 + React 19 web application front-end for LaunchKit, configured with Tailwind CSS v4 and Framer Motion.

## Directory Structure

- `src/app/`: App router pages, layouts, and global styles.
- `src/components/`: Client-side components.
- `public/`: Static assets.

## Commands

Run directly inside this directory or from the root workspace using filter commands:

- `pnpm dev`: Start development server.
- `pnpm build`: Compile production app bundle.
- `pnpm lint`: Lint codebase using next-lint.
- `pnpm typecheck`: Validate TypeScript compiler checks.
