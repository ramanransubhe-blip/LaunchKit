# Monorepo Folder Structure

Overview of workspace structure:

- `/apps/web`: Primary Next.js 16 SaaS web application.
- `/packages/ui`: Design tokens, Framer Motion animations, and shadcn components.
- `/packages/api`: Standardized REST envelopes and Developer SDK client.
- `/packages/cli`: Command-line interface code generators.
- `/packages/database`: Drizzle schema, indices, migrations, and model triggers.
