# LaunchKit Documentation

This directory contains technical guides, architectural details, and user manuals for the LaunchKit project.

## Document Listing

- `architecture/`: Deep dive into how packages, UI design tokens, and applications compile and communicate.
- `deployment/`: Guides for setting up Vercel, Docker registries, and AWS pipelines.
- `guides/`: Quickstarts for billing setup, database migrations, and auth setups.
